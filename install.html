ACFUN补全计划 - 屏蔽插件

本插件原AC投稿被删，猴子屁眼小。

支持浏览器：Chrome / FireFox / 360极速 / QQ / Edge / 遨游 / 等其他chromium内核支持插件系统的浏览器

安装方法如下

第一步

安装浏览器脚本插件: TamperMonkey / VioelentMonkey

具体请到浏览器的[扩展工具]/[应用中心] 等地方搜索安装

Chrome如果没办法翻墙安装插件,可以到 https://www.chrome666.com/chrome-extension/tampermonkey.html 下载插件. 然后在Chrome的扩展中心右上角,打开[开发者模式],将下载解压的crx文件拖入扩展中心页面即可.

注意：老版遨游浏览器可以考虑ViolentMonkey

第二步

安装ACFUN屏蔽插件, 点击一下任意链接安装:

https://greasyfork.org/scripts/381476-acfun%E5%B1%8F%E8%94%BD%E8%AE%A1%E5%88%92/code/Acfun%E5%B1%8F%E8%94%BD%E8%AE%A1%E5%88%92.user.js

https://github.com/baldhumanity/AcfunComplement/raw/master/acfunBlock.user.js
